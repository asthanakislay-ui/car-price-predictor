# 🚗 Car Price Predictor

A Machine Learning web app that predicts used car prices based on user inputs.

## 🔥 Features

* Predicts car price using Linear Regression
* Cleaned dataset
* Simple UI

## 🛠 Tech Stack

* Python
* Scikit-learn
* Pandas, NumPy
* Streamlit

## 🚀 How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
